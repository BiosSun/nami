提供一个页面级的水平布局导航栏组件。

{@demo "./demos/common.jsx"}

## 参数

### Navbar

| 参数       | 说明         | 类型                         | 默认值 |
| ---------- | ------------ | ---------------------------- | ------ |
| `children` | 导航栏内容项 | `Navbar.Brand` &#124; `Menu` |        |

### Navbar.Brand

| 参数       | 说明           | 类型                  | 默认值 |
| ---------- | -------------- | --------------------- | ------ |
| `children` | 标识文本或图像 | `string` &#124; `img` |        |
